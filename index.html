6. Listas: entenda as tag ol, ul, li

As tags ol, ul e li, são usadas para criar listas, tipo aquele índice que foi criado lá em cima

Com a tag ol podemos criar listas ordenadas de forma numerada.

  1. Item 1
  2. Item 2
  3. Item 3

E com a tag ul criamos listas não ordanadas que ficam com essa marca de uma bolinha.

  • Item A
  • Item B
  • Item C

E a tag li é usada para definir cada item individual dentro da lista
